# Certifyied Reviews & Feedback Funnels

This repository contains the standalone portals for Certifyied Reviews:

1. **`reviewdash/`**: The main administrator console and customer reviews feedback funnel.
2. **`clientReview/`**: The light-themed client dashboard where business owners can log in and view their respective customer reviews log.

## Features
- Clean light theme.
- Passwordless magic link email authentication.
- Automatic local-to-live API worker fallback.

## Local Development
To run any of the portals locally:
1. Navigate to the project folder (`clientReview` or `reviewdash`).
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the local Vite development server.
